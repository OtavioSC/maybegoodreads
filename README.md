## How to run Backend
Navigate into the directory

    cd packages/server

Install all dependencies

    yarn install
    
Create .env file and fill all the variables

    mv .env-example .env

Run the server

    yarn dev
    

## How to run Frontend
Navigate into the directory

    cd packages/web

Install all dependencies

    yarn install

Run the server

    yarn dev
